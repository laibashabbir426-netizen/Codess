# AS25 - Luxury Car Web Application

A responsive and interactive web-based project developed as part of the AICT course for the Spring 2026 semester at Air University Islamabad, Department of Computer Science.

## 🚀 Project Overview
AS25 is a modern web application designed for car enthusiasts and buyers looking to explore, compare, and manage luxury vehicles. The platform features an intuitive UI, smooth responsiveness across devices, and interactive components to enhance user engagement.

## 🛠️ Tech Stack Used
* **HTML5:** For structuring the web pages.
* **CSS3 & Bootstrap:** For custom styling, theme implementation, and ensuring a fully responsive mobile/desktop view.
* **JavaScript (ES6):** For adding interactive features and dynamic logic.

## 📁 Project Structure & Key Files
Based on the project build, the repository contains the following architecture:

*   `index.html` - The main landing page introducing the luxury car platform.
*   `cars.html` - A dedicated catalog showcasing available luxury cars.
*   `compare.html` - An interactive page allowing users to compare different car specifications.
*   `dashboard.html` - A user/admin dashboard tracking platform metrics and user activities.
*   `css/`
    *   `desktop-view.css` - Custom styling optimized for larger screens.
    *   `luxury.css` - Theme-specific aesthetics catering to the luxury branding.
*   `js/`
    *   `app.js` - Main JavaScript file driving the interactivity and functionality of the application.

## ✨ Core Features
*   **Fully Responsive Design:** Seamlessly transitions between desktop displays and mobile interfaces using Bootstrap and custom media queries.
*   **Interactive Comparisons:** Dynamically compare specifications of selected luxury vehicles on the comparison page.
*   **Modern Dashboard UI:** A clean layout representing user insights and car data effectively.
